# 🌟 Viction Retrodrop Season 2 Ideabook

A visionary blueprint for how each Retrodrop recipient could use their VIC allocation to build meaningful, Viction-native experiences that bring glory to users, culture, and builders.

---

## 🚀 Project 1: Coin98 Super Wallet

**Allocation**: 472,500 VIC
**Category**: Core Project

### 🔮 Idea: **Viction Passport — The Onchain Identity Layer**

**Concept**:
Create an NFT-based "Viction Passport" that represents a user's identity, culture, and reputation on Viction. Built into Coin98's wallet, it combines:

* Verified identity (OneID)
* NFT holdings (Dagora)
* Onchain reputation (Efihub)
* DAO activity or social trust (Ivy)

**Features**:

* Dynamic NFT viewable in Coin98
* Reputation score embedded onchain
* Unlocks gated access to dApps or experiences
* Upgradable via VIC staking or contributions

**Ecosystem Touchpoints**:

* OneID ✉️
* Dagora 🎨
* Efihub ✨
* Ivy 🗳️

**Impact**:

* Turns Coin98 into the gateway for Viction's cultural layer
* Encourages onchain action with visible identity rewards
* Enhances composability across the ecosystem
